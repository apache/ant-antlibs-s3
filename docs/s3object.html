s3:object

Description

Ant resource representing an S3 object or object version (an object version will ordinarily come from s3:objects). Supports a client although in normal usage this is unnecessary.

Parameters

Attribute Description Required
bucket S3 bucket Yes
key S3 key Yes
clientrefid Ant reference ID of an s3:client defined elsewhere No
contenttype MIME type of this resource No

Parameters specified as nested elements

client

s3:client

metadata

Inline properties specifying S3 object metadata

tagging

Inline properties specifying S3 object tagging
