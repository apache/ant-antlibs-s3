s3:client

Description

Declare an Amazon S3 client for use in the S3 Antlib.

Parameters specified as nested elements

builder

The builder element applies its XML configuration to an AWS SDK 2.x S3ClientBuilder. Attributes are interpreted as simple string-convertible properties, while elements are interpreted as nested builders of the type specified by the method parameter.

credentials

The credentials element permits an easier mechanism for specifying client credentials.

Parameters
Attribute Description Required
accesskey AWS Access key Yes, unless profile is specified
secretkey AWS Secret key Yes, unless profile is specified
profile AWS profile ID Yes, unless @accesskey and @secretkey are specified

http

Inline properties specifying SdkHttpConfigurationOption constants
