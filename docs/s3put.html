s3:put

Description

Puts nested resources into an S3 bucket.

Parameters

Attribute Description Required
bucket destination bucket Yes
clientrefid Ant reference ID of an s3:client defined elsewhere Yes, unless a nested s3:client is specified
enablemultiplemappings Enable multiple destinations generated from nested mapper elements No, default false
filtering Whether to apply global Ant filters No, default false
inputencoding Input encoding No
outputencoding Output encoding No

Parameters specified as nested elements

resource collection

Any number of nested resource collections are accepted.

FilterSet

Any number of nested filtersets are accepted.

FilterChain

Any number of nested filterchains are accepted.

Mapper

Any number of nested mapper elements are accepted.

FileNameMapper

Any number of type defined mappers are accepted.
