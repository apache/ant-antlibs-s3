s3:objects

Description

Ant resource collection of s3:object resources.

Parameters

Attribute Description Required
bucket S3 bucket Yes
delimiter S3 delimiter No, default /
cache Whether to cache discovered resources No, default true
casesensitive Whether the resource search is performed in a case-sensitive manner No, default true
as object precision: object|version No, default object
includeprefixes Whether to include s3 prefix objects as directory resources No, default false
clientrefid Ant reference ID of an s3:client defined elsewhere Yes, unless a nested s3:client is specified
includes comma-delimited include patterns No
includesfile file containing line-delimited include patterns No
excludes comma-delimited exclude patterns No
excludesfile file containing line-delimited exclude patterns No

Parameters specified as nested elements

client

s3:client

patternset

Any number of nested patternsets are permitted.

include

Any number of nested include elements are permitted.

exclude

Any number of nested exclude elements are permitted.

selector

Any number of nested resource selectors are permitted and applied in OR fashion.
